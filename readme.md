## machine Requirements for laravel 5.5 

- PHP >= 7.0.0
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- Tokenizer PHP Extension
- XML PHP Extension

## Beer Wiki Installation steps

- composer install
- npm install
- npm run prod
- php artisan serve
- **[Open the app](http://localhost:8000/)**
